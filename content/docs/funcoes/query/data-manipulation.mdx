# Funções de Manipulação de Dados

A linguagem da função QUERY suporta diversas funções para agregar e transformar dados diretamente na consulta.

| Função | Descrição |
|---|---|
| `count()` | Conta os registros não-nulos |
| `sum()` | Soma os valores numéricos |
| `avg()` | Calcula a média |
| `min()` | Retorna o menor valor |
| `max()` | Retorna o maior valor |

> Estas funções são usadas em conjunto com a cláusula `group by`.
