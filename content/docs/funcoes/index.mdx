# Funções

Referência das principais funções do Google Sheets abordadas neste handbook.

## Funções disponíveis

- **[QUERY](/docs/funcoes/query)** — a mais importante; usa uma linguagem semelhante a SQL
- **[XLOOKUP](/docs/funcoes/xlookup)** — busca avançada, substituto do VLOOKUP
- **[MAP](/docs/funcoes/map)** — aplica uma lambda a cada elemento de um array
- **[ARRAYFORMULA](/docs/funcoes/arrayformula)** — expande fórmulas para um intervalo inteiro
- **[GOOGLEFINANCE](/docs/funcoes/googlefinance)** — dados financeiros em tempo real
